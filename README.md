# Button Clicker

A simple button clicker built with HTML, CSS, and JavaScript to practice DOM manipulation and event handling.

## Features

* Increment a counter by clicking the button
* Reset the counter to zero
* Interactive button press animations
* Styled user interface using CSS Flexbox

## Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

The application stores a counter value in JavaScript. Each click of the main button increases the counter and updates the displayed value on the page using DOM manipulation
The reset button sets the counter back to zero.

## What I Learned

This project helped me practice:
* Selecting DOM elements with `querySelector`
* Adding event listeners
* Updating content with `textContent`
* Managing state with variables
* Styling interactive buttons with CSS
